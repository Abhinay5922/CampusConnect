# CampusConnect (MERN Starter)

This is a starter full-stack project for CampusConnect â€” a placement portal connecting alumni and students.

## Structure
- backend: Express + MongoDB + Socket.io
- frontend: Vite + React

## Quick start
1. Backend:
   - cd backend
   - cp .env.example .env (set MONGO_URI and JWT_SECRET)
   - npm install
   - npm run dev

2. Frontend:
   - cd frontend
   - npm install
   - npm run dev

Frontend expects backend API at http://localhost:5000 by default.
